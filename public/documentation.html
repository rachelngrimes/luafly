Luafly Documentation
Luafly is a complete whitelist management system for Roblox script developers. Control who can run your scripts using HWID-based authentication with key redemption, status checks, and automatic resets.
ComponentPurposeURL
LuaflyWhitelist backend, dashboard, key management, execution logsluafly.vercel.app
Protected Script HostServes script payloads after HWID verificationinternal hosting
You need to be whitelisted before running any scripts. Contact the developer or redeem a key to get access.

How it works

The full flow from getting access to running the script.

1
Get a key from the developer
Keys look like LUAFLY-XXXXXXXX-XXXX-XX and are one-time use. Each key has a duration — 7 days, 30 days, or lifetime.
2
Redeem the key
Run the redeem script in your executor with your key. This registers your HWID in the database and marks you as whitelisted.
3
Run the loader
Every time you want to use the script, run the loader. It checks your HWID and loads the script if you're whitelisted.
4
Script loads
If your HWID is valid, the script loads automatically. If not, you get kicked with a message explaining why.

Whitelist system

The whitelist is HWID-based. Your Hardware ID is tied to your device and checked every time you run the loader.

StatusResult
whitelistedScript loads successfully
blacklistedKicked with ban reason shown
not foundKicked — not in the whitelist
expiredKicked — whitelist period ended

Redeeming a key

Run this once in your executor to redeem your key and get whitelisted. Replace the placeholder key with the key you received.

lua
-- Paste your key (global variable)
key = "LUAFLY-XXXXXXXX-XXXX-XX"
loadstring(game:HttpGet("https://luavex.localplayer.dev/files/v4/loaders/18e4db09b0e7289fdc4d463d874c6480"))()
After running this you should see Successfully redeemed! You are now whitelisted. in the console. Keys are one-time use.

Key durations

DurationAccess period
7day7 days from redemption date
30day30 days from redemption date
lifetimeNever expires

Running the loader

Run this every time you want to use the script. It verifies your HWID and loads the script if you're whitelisted.

lua
loadstring(game:HttpGet("https://luavex.localplayer.dev/files/v4/loaders/782cb7b5d699cf8c1de92e4391b9c449"))()
The loader shows a brief loading screen while it verifies your HWID. This takes 1–2 seconds.

What happens when you run the loader

1
HWID verification
Your Hardware ID is sent to the Luafly API and checked against the whitelist database.
2
Script load
If whitelisted, the script is fetched from the protected host and executed. The script source is never exposed.

Checking your status

Run this at any time to check your whitelist status, expiry, resets left, and token balance.

lua
loadstring(game:HttpGet("https://luavex.localplayer.dev/files/v4/loaders/67869cdace6d99e9a20fabf4034da5ac"))()
The status script prints your current whitelist status, expiration date, available HWID resets, and token balance.

HWID reset

If you get a new PC or reinstall Windows, your HWID changes. Use your reset tokens to update your HWID in the system.

Resets cost 1 token each. If you don't have any tokens left, contact the developer (they can manually reset your HWID or add tokens).
Don't have your old HWID? No problem — just ask the developer for help. They can reset your HWID manually from the dashboard without needing the old HWID.
lua
-- Run on your new device. Replace old_hwid with your previous HWID (if known).
local HttpService = game:GetService("HttpService")
local old_hwid = "YOUR-OLD-HWID-HERE"
local new_hwid = game:GetService("RbxAnalyticsService"):GetClientId()
local url = "https://luafly.vercel.app/api/hwidreset?old_hwid=" .. old_hwid .. "&new_hwid=" .. new_hwid
local data = HttpService:JSONDecode(game:HttpGet(url))
print(data.success, "Tokens left:", data.tokens_left)

If you don't remember your old HWID: simply contact the developer via Discord. They can manually update your HWID or add reset tokens to your account.

Dashboard

The dashboard at luafly.vercel.app is password protected and only accessible by the developer.

TabWhat it shows
OverviewUser counts, executions today, recent users, live logs, 7-day execution graph
UsersFull user list with search, sort, and CSV export
KeysGenerate and manage whitelist keys
LogsFull execution history filterable by today or this week

Managing users

Users can be added manually from the dashboard or automatically when they redeem a key.

FieldDescription
hwidHardware ID from RbxAnalyticsService
usernameRoblox username or any label you assign
statuswhitelisted, blacklisted, or expired
ban_reasonShown in the kick message when blacklisted
notesPrivate notes only visible on the dashboard
hwid_resets_leftNumber of remaining HWID resets
tokensReset token balance — each reset costs 1
executorAuto-detected when they run the loader
last_seenLast time they successfully executed the script

Keys

Keys let you give access without manually adding someone's HWID. Generate them from the dashboard or via the Discord bot.

Keys are one-time use. Once redeemed the key is permanently used. Generate a separate key for each person.
DurationAccess period
7dayExpires 7 days after redemption
30dayExpires 30 days after redemption
lifetimeNever expires

Logs

Every action is logged automatically and visible from the Logs tab on the dashboard.

ActionTriggered when
executedSomeone successfully loads the script
redeemed keySomeone redeems a whitelist key
hwid resetSomeone resets their HWID

Discord bot commands

The Luafly bot lets you manage users directly from Discord. All commands are restricted to the developer only.

CommandDescription
/pingCheck if the bot is online
/whitelist hwid usernameWhitelist a user by HWID
/blacklist hwid reasonBlacklist a user — creates entry if they don't exist yet
/removeuser hwidRemove a user from the database entirely
/genkey durationGenerate a key (7day, 30day, lifetime)
/addtokens hwid amountGive a user HWID reset tokens
/userinfo hwidView all info about a user
/statsShow overall stats across all users and logs
